The Loop

How SocraticRL Works

A closed-loop RL environment. The agent asks, the student responds, rewards flow, the agent learns — all locally, in real time.

01

Episode Reset

A student scenario is randomly chosen — a student profile, topic, and misconception are loaded. The student opens with their confusion.

02

Agent Questions

The trained LLM agent generates a Socratic question — probing, not answering. Chain-of-thought reasoning is internal, never exposed to the student.

03

Student Responds

The local student simulator generates a realistic response based on current understanding, misconception, and the agent's question quality.

04

Reward Computed

Multi-component reward: question quality, topic relevance, understanding delta, and terminal success bonus. Guides the agent toward optimal teaching strategies.

05

Terminal or Continue

Episode ends when understanding ≥ 90% or max turns reached. Efficiency bonus rewards agents who achieve mastery in fewer turns.
